# 大数据分析平台可视化大屏

一个基于 Vue 3 + Vite 的科技风数据可视化大屏项目，包含设备分类、设备接入趋势、设备状态、设备收发数据、告警数据、大事记、快速导航和中心园区可视化场景。

## 特性

- Vue 3 `<script setup>` 单文件组件开发
- 固定大屏画布并根据窗口等比缩放
- 设备分类环图与随动数据标牌动画
- 科技粒子、扫描光、数据光柱和面板流光效果
- 纯 CSS 图表与视觉动画，无额外图表依赖
- MIT 开源协议

## 本地运行

```bash
npm install
npm run dev
```

访问开发服务输出的本地地址，默认通常为：

```text
http://127.0.0.1:5173/
```

## 构建

```bash
npm run build
```

构建产物会输出到 `dist/` 目录。

## 技术栈

- Vue 3
- Vite 5
- CSS3 Animation
